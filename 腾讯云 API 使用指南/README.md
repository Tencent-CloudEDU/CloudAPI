## 课程简介

云API是腾讯云开放生态的基石。通过API，我们可以以接口的形式访问腾讯云的各类资源，本套视频主要通过简单常用的例子，为大家分别讲解了腾讯云云API的使用方法，SDK的使用方法以及CLI的使用方法。
